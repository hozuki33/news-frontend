# Vue 3 + Vite

项目简介

本项目是一个完整的新闻系统前端，实现了新闻内容的展示与管理，采用前后端分离架构，前端负责页面展示与交互，后端提供 RESTful API 接口。


功能模块

首页新闻分类展示（今日要闻 / 游戏快报 / 动画资讯 / 通知公告）

新闻轮播图展示（今日要闻）

新闻列表分页展示

新闻详情页（支持富文本内容渲染）

点击标题 / 卡片跳转新闻详情


技术栈

Vue 3

Vite

Element Plus

Vue Router

Axios

WangEditor（富文本编辑器）
